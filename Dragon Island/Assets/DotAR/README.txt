This asset is a simple AR library for dot marker. Designed for ease of use.

[how to use]

1. Print out a marker or display it. (Sample/DotAR_marker.pdf)

2. Connect a camera

3. Set marker parameter. (Sample/DotAR_sample.cs)

   // for example, dot:6x6, distance:1.0
   dotar.setMrk (6, 6, 1.0);

3. Start DotAR_sample scene
